Executive Summary

Making spatial irrigation intelligence actionable.

Spatial Adjust integrates Toro's TurfRad sensors into an AI-assisted map and recommendation UX. While sensors gathered millions of data points on soil moisture, no system existed to help superintendents make sense of that data at scale.

  • Outcome: Seamlessly bridged autonomous recommendations with human oversight.
  • Key Challenge: Designing "override trust patterns" so experts felt comfortable delegating decisions to an algorithm without losing control.
  • Focus: AI-enabled workflows, hardware-to-software translation, and spatial reasoning UX.

Spatial Adjust translates millions of TurfRad sensor data points into an actionable, override-friendly map interface.

The Problem

Drowning in Telemetry

Hardware sensors like TurfRad can gather immense volumes of soil moisture data. However, for a golf course superintendent, raw data isn't enough. They need to know what action to take. When an algorithm recommends a severe watering reduction across a 50-acre property, the user's immediate instinct is distrust. The system must explain "why" and offer an easy "override" path.

Key Decisions

Override Trust Patterns

1. Algorithmic Transparency

We designed the UI to "show its work." Instead of a blackbox recommendation, Spatial Adjust highlights the specific TurfRad sensor logs and weather forecasts that triggered the recommendation.

2. The Confident Override

If an expert disagrees with the AI, overriding the decision is a frictionless one-click action. This builds trust, establishing AI as an assistant rather than an unyielding authority.

3. Spatial Reasoning UX

Data was plotted spatially across the course map, transforming abstract tables into visual heatmaps of soil moisture deficiencies.
